1 One-Time Setup: Install tmate

Open Terminal (press Cmd + Space, type "Terminal", press Enter)

brew install tmate

Open Terminal

sudo apt update && sudo apt install -y tmate
2 When You Need Help

Run this command in Terminal:

tmate

Look at the bottom of the screen for a line like:

ssh abc123xyz@nyc1.tmate.io
3 Send the Connection

Copy that entire line and send it via Signal or iMessage.

⚠️ Security Notice

Never send the connection via email, SMS, or Slack. Only use encrypted messaging (Signal or iMessage).

4 Watch & End

You'll see everything happening in real-time. When we're done:

exit

Or just close the Terminal window. The session ends immediately.
